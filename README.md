## Software List
- Compositor: niri
- Bar: waybar
- Wallpaper daemon: swww
- Menu: vicinae
- Idle manager: hypridle
- Screen locker: hyprlock
- Notifications: swaync
- Volume/brightness control: swayosd
- Terminal: alacritty
- Shell: zsh
- Shell prompt: oh-my-posh
